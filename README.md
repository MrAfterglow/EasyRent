# 随心租
This Is a new great project, use Flask and ajax.



### 项目概述：



		项目名为随心租，一个租房类web应用，与市场上现有的项目不同的是此项目是同时面向租客与房东的，每一位用户既可以作为租客，也可以发布房源。

##### 框架：

	web后端使用Python_flask框架，与前端前后分离，

##### API：

	前端使用ajax请求数据，后端返回json数据。

##### SQL：

	使用MySQL存储数据
	
	使用Redis存储cookie与session
